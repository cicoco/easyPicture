# EasyPicture 产品需求文档（PRD）

**版本**：v1.0  
**日期**：2026-03-09  
**状态**：草稿

---

## 一、产品概述

### 1.1 产品定位

EasyPicture 是一款轻量级本地桌面图片编辑工具，面向需要快速完成基础修图操作的普通用户。无需上传云端，全程本地处理，图片质量完整保留。

### 1.2 目标用户

- 需要快速裁剪、旋转图片的普通用户
- 需要抠图（去除背景）但不想依赖在线工具的用户
- 对图片隐私有顾虑、不希望上传至云端的用户

### 1.3 核心价值

| 价值点       | 说明                                  |
|------------|---------------------------------------|
| 即开即用     | 安装后双击打开，无需注册/联网            |
| 图片无损     | 全程使用原始像素数据，导出支持无损格式   |
| 完全本地     | 所有处理在本机完成，不上传任何数据       |
| 轻量快速     | 应用体积控制在 100MB 以内，操作响应流畅  |

## 二、功能需求

### 2.1 功能清单

| 功能编号 | 功能名称       | 优先级 | 说明                                 |
|--------|--------------|--------|--------------------------------------|
| F-01   | 导入图片       | P0     | 支持从本地选择图片文件打开              |
| F-02   | 导出图片       | P0     | 支持将编辑结果保存至本地               |
| F-03   | 裁剪          | P0     | 通过鼠标拖拽框选区域，裁取该区域        |
| F-04   | 区域选择       | P1     | 框选或套索选定图片中的任意区域          |
| F-05   | 删除选区内容   | P1     | 将选中区域内容清除（填充透明或背景色）   |
| F-06   | 抠图          | P1     | 自动/半自动去除图片背景，生成透明背景图  |
| F-07   | 旋转图片       | P0     | 支持 90°/180°/270° 旋转及任意角度旋转  |
| F-08   | 符合画布       | P1     | 自动裁去图片四周多余的透明像素，保留主体内容的最小边界框 |
| F-09   | 缩放到指定尺寸 | P1     | 将图片重采样到精确的宽高像素值，可选锁定宽高比 |

> P0：必须上线；P1：首版包含

### 2.2 功能详细描述

#### F-01 导入图片

- 支持格式：PNG、JPG/JPEG、BMP、TIFF、WEBP
- 交互方式：
  - 菜单栏「文件 → 打开」
  - 快捷键 `Cmd+O`（macOS）/ `Ctrl+O`（Windows/Linux）
  - 拖拽图片文件到窗口
- 打开后图片自适应显示在画布区域（不缩放原始数据，仅视图缩放）
- 一次只处理一张图片，打开新图前提示是否保存当前修改

#### F-02 导出图片

- 支持格式：PNG（无损）、JPG（可设质量 0-100，默认 95）、BMP、TIFF
- 交互方式：
  - 菜单栏「文件 → 导出」
  - 快捷键 `Cmd+S` / `Ctrl+S`
- 导出时弹出格式和路径选择对话框
- 如果图片含透明通道，强制建议使用 PNG 格式
- 原始图片不被覆盖（另存为逻辑）

#### F-03 裁剪

- 用户在画布上拖拽鼠标框选裁剪区域
- 拖拽时显示半透明蒙层，突出选区
- 支持调整选区边角大小
- 确认裁剪后，画布内容替换为裁剪区域，画布尺寸随之变化
- 支持撤销（`Cmd+Z` / `Ctrl+Z`）

#### F-04 区域选择

- 提供两种选择工具：
  - **矩形选框**：拖拽绘制矩形选区
  - **套索工具**（可选，首版可省略）：自由绘制多边形选区
- 选区显示虚线边框（蚂蚁线效果）
- 按 `Esc` 取消选区
- 选区可在工具切换后保持，直至主动取消

#### F-05 删除选区内容

- 前提：已有选区（F-04）
- 操作：按 `Delete` 或 `Backspace` 键，或点击工具栏「删除选区」按钮
- 若图片支持透明（PNG/TIFF）：将选区填充为透明
- 若图片不支持透明（JPG/BMP）：将选区填充为白色（或用户设置的背景色）
- 操作后选区保留，支持撤销

#### F-06 抠图（背景去除）

- 方式：半自动 GrabCut + 后处理流水线
- 流程：
  1. 用户在工具栏选择「抠图」工具
  2. 用鼠标框选需要保留的主体区域（粗略框选即可，框大一点效果更好）
  3. 点击「执行抠图」按钮，后台线程处理，显示进度条
  4. 处理流程：GrabCut 图割 → 形态学修复 → 连通域过滤 → 双线性放大 → 边缘羽化
  5. 背景区域 alpha=0（透明），主体边缘有软过渡，结果显示在画布（棋盘格背景）
  6. 用户可继续用「选框 → 删除」手动修补不满意的区域
- 结果自动转为含透明通道的图像，建议导出为 PNG
- 支持撤销
- **局限性**：头发丝、毛发、半透明物体边缘质量有限（GrabCut 算法固有局限）；背景越纯净、与主体颜色对比越强，效果越好

#### F-07 旋转图片

- 快捷旋转：
  - 顺时针 90°：工具栏按钮或 `Cmd+]`
  - 逆时针 90°：工具栏按钮或 `Cmd+[`
  - 180°：菜单选项
- 旋转后画布尺寸随图片方向自适应更新
- 支持撤销

#### F-08 符合画布（Trim to Content）

- **自动**检测图片中所有不透明像素（alpha > 阈值，默认 0）的边界框
- 将图片裁切为该边界框，去除四周多余的透明像素，保留主体内容
- 若整张图片均为不透明，提示用户"无需裁切"，不修改图像
- 典型使用场景：
  - 抠图（GrabCut）后背景变透明，一键去除边缘空白区域
  - 导出前压缩画布到最小尺寸，减小文件体积
- 支持撤销
- 交互入口：菜单栏「图像 → 符合画布」

> **本质**：等同于 Photoshop「图像 → 裁切（Trim）→ 基于透明像素」。操作无需用户输入参数，一键执行。

#### F-09 缩放到指定尺寸（Resize to Size）

- 弹出对话框，用户输入目标**宽度**（W）和**高度**（H）
- **锁定宽高比**选项（默认开启）：输入一侧时，另一侧自动按比例计算
- **重采样插值算法**：
  - 缩小：LANCZOS4（Sinc 滤波，最高质量，避免摩尔纹）
  - 放大：CUBIC（双三次插值，平滑）
- 支持按百分比缩放（50% = 半尺寸）
- 支持撤销
- 交互入口：菜单栏「图像 → 缩放图片...」

## 三、非功能需求

### 3.1 性能要求

| 操作              | 目标响应时间    |
|-----------------|--------------|
| 打开图片（≤20MB） | < 1 秒        |
| 裁剪/旋转         | < 0.5 秒      |
| 符合画布 / 缩放    | < 1 秒        |
| GrabCut 抠图      | < 5 秒（视图片大小）|
| 导出图片          | < 2 秒        |

### 3.2 兼容性

- 操作系统：macOS 12+，Windows 10+（Linux 兼容但不作为主要目标）
- 显示分辨率：支持 1280×720 及以上，适配 HiDPI/Retina 屏

### 3.3 图片质量保证

- 内存中始终保持原始分辨率的图像数据（numpy array，uint8）
- 视图缩放仅为显示用途，不影响实际像素
- 导出时直接写入原始数组，不经过额外压缩/重编码（PNG 模式）

├──────────┬──────────────────────────────────────┤

- 工具栏图标使用系统图标或 Material Icons
- 选中工具高亮显示

- 画布支持 `Ctrl/Cmd + 滚轮` 缩放（视图缩放，不改变图片）
- 画布支持按住空格键拖动平移

## 五、边界与限制（首版 v1.0）

- 不支持多图层编辑
- 不支持文字添加、滤镜、色彩调节（后续版本扩展）
- 不支持撤销历史超过 20 步
- 不支持批量处理
- 抠图使用 GrabCut + 形态学/羽化后处理，不含 AI 深度学习模型
- 不支持头发丝/毛发级别的精细抠图（需 Alpha Matting 或 AI 模型，规划于 v1.3）
